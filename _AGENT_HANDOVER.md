# AI Agent Handover: Dragon Delves Wiki Maintenance

This document provides the standardized workflow and formatting rules for maintaining the **Dragon Delves** campaign chronicle. Use this to ensure continuity across all future sessions.

## üåü The "Premium" Formatting Standard

Every session log should follow the "Session 1 Golden Standard." The structure MUST be as follows:

1.  **Headers**: Use `# Chapter [Ordinal]: [Title]` for the main heading.
2.  **Session Cheat Sheet**: A high-level summary section at the top including:
    - **Dramatis Personae**: NPCs encountered.
    - **Key Items & Quests**: Loot and goals.
    - **Locations & Factions**: Relevant world-building.
    - **Current Objective**: The immediate next step.
3.  **Narrative Style**:
    - **Entity Bolding**: Always **bold** the first mention of Character Names, Key Items, and Locations.
    - **Dividers**: Use `---` between the cheat sheet and the narrative.
    - **Combat Markers**: Demarcate combat with `**_Combat Starts_**` and `**_Combat Ends_**`.
    - **Vital Discoveries**: Highlight key lore with `**[Vital Discovery]**`.
4.  **Navigation Trails**: Every file must end with a footer linking to the previous session, the Master Index, and the next session.
